# CLIP-ReID 跨模态行人重识别论文复现
## 项目简介
本项目为南昌大学AI课题组招新考核项目，复现CCF A类AAAI顶会论文CLIP-ReID，基于PyTorch完成跨模态行人重识别任务。

论文地址：https://ojs.aaai.org/index.php/AAAI/article/view/25225
开源代码：https://github.com/Syliz517/CLIP-ReID

## 完成考核内容
1. 搭建CUDA+PyTorch独立显卡深度学习环境
2. 正常跑通1-2个ReID行人重识别数据集
3. 模型两阶段训练完整运行，无需跑完全部epoch即可正常收敛
4. 可选优化：修改、替换损失函数，对比实验效果
5. 完成模型推理Demo可视化展示
6. 撰写完整项目实验结果报告

## 技术环境
Python、PyTorch、CUDA、Linux、Git
